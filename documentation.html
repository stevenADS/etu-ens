C'est quoi une API ?

Une API pour application programming interface permet a deux applications de communiquer entre elles

Dans le cadre de notre projet l'api permet de recuperer des informations sur chacun des etudiants inscrit grace aux formulaires.

les liens de l'API se trouve sur : https://steven-adji.yo.fr
