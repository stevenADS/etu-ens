Notre Api a pour but de récupérer les information sur les ettudiants.

Pour se faire il faut se rendre sur le lien API du site :

URl:https://steven-adji.yo.fr/api.php

Les etudiants ont comme information:

Filliere MIPI/LPI-1
Nom Prenom
Groupe A1/A2/LPI...
Mail exemple@gmail.fr

Requêtes

Pour récupérer par exemple la liste des Fillieres :

  • Requête: https://steven-adji.yo.fr/api.php?filliere="la filliere" & key=n°de la clé


    Pour récupérer par exemple la liste des Groupes :

  • Requête: https://steven-adji.yo.fr/api.php?filliere="le Groupe" & key=n°de la clé


    Pour récupérer par exemple la liste des Etudiants :

  • Requête: https://steven-adji.yo.fr/api.php?filliere="Etudiant" & key=n°de la clé

    Pour récupérer le TOUT: La filliere/Groupes/Etudiants
